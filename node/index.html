  • 由路由和中间件组成，本质上可以理解为应用在调用各种中间件

# 中间件

中间件（Middleware）是一个函数，它可以访问请求对象 req，响应对象 res，以及 web 应用中处于请求 - 响应循环流程中的中间件 next。

如果当前中间件没有结束循环，则必须调用 next()把控制权交给下一个中间件，否则请求会挂起。

# 主要功能

  • 执行任何代码
  • 修改请求和响应对象
  • 终结请求-响应循环
  • 调用堆栈中的下一个中间件
